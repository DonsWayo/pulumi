Get started with deploying modern web applications to AWS. This guide walks you through the setup and deployment process.

## Overview

These Pulumi packages simplify deploying Next.js, SvelteKit, and React Router applications to AWS using serverless architecture. Deploy production-ready applications with automated infrastructure provisioning.
